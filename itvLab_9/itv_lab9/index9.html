1. Дараах кодоос алдаа ол. Нэгээс олон алдаа байж болзошгүй.
a. for (x= 100, x >= 1, ++x )
document.writeln( x );
алдаа нь зарлах кодын толгой хэсэг ; буюу semicolon тавина. Мөн --x байх ёстой.

b) Дараах код value хувьсагчийн тэгш сондгой эсэхийг хэвлэнэ.
switch ( value % 2 ) {
case 0:
document.writeln( "Even integer" );
case 1:
document.writeln( "Odd integer" );
Алдаа нь default утга байхгүй , зарлах value

c) Дараах код 19-оос 1 хүртэлх сондгой тоонуудыг хэвлэнэ:
for (x= 19; x >= 1; x += 2 )
document.writeln( x );
Алдаа нь x -= 2; гэж болно, x- ээ зарлаж өгнө, x - ээ 1 ээс эхлүүлээд өсөх дарааллаар хэвлэж болно.

d) Дараах код 2-оос 100 хүртэлх тэгш тоонуудыг хэвлэнэ:
var counter = 2;
do {
document.writeln( counter );
counter += 2;
} while ( counter < 100 );
Алдаа нь 100 хүртэлх гэхээр counter <= 100 гэж бич
